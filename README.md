# Sisain.online - Marketplace Surplus Pangan (GEMASTIK 2026)

Sisain.online adalah platform marketplace berbasis Progressive Web App (PWA) yang menghubungkan merchant surplus pangan dengan konsumen terdekat.

## 🚀 Tech Stack
- **Frontend**: React.js (Vite)
- **Icons**: Lucide React
- **Styling**: Neumorphism (Custom CSS)
- **Deployment Plan**: Vercel / Netlify

## 📂 Struktur Project
- `/src`: Berisi komponen React (`App.jsx`), entry point (`main.jsx`), dan styling.
- `index.html`: File entry point untuk browser.
- `package.json`: Definisi dependensi project.
- `vite.config.js`: Konfigurasi build tool Vite.

## 🛠️ Langkah Menjalankan (Local Development)
1. Pastikan Anda memiliki **Node.js** terinstal.
2. Buka terminal di folder ini.
3. Jalankan `npm install` (Membutuhkan sekitar 300-400MB ruang disk).
4. Jalankan `npm run dev` untuk memulai server development.
